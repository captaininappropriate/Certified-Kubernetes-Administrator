# A collection of manifests for the different type of objects within a Kubernetes environment

1. Core Workloads
    - Pod
    - Deployment
    - DaemonSet
    - StatefulSet
    - ReplicaSet

2. Networking
    - Service
    - Ingress
    - NetworkPolicy
    - EndpointSlice
    - Endpoints

3. Identity & Access
    - ServiceAccount
    - Role
    - RoleBinding
    - ClusterRole
    - ClusterRoleBinding

4. Configuration & Storage
    - ConfigMap
    - Secret
    - PersistentVolume (PV)
    - PersistentVolumeClaim (PVC)
    - StorageClass
    - VolumeAttachment

5. Cluster Administration
    - Namespace
    - Node
    - ResourceQuota
    - LimitRange
    - PriorityClass

6. Autoscaling & Scheduling
    - HorizontalPodAutoscaler (HPA)
    - PodDisruptionBudget (PDB)
    - RuntimeClass

7. Policy & Security Controls
    - PodSecurityPolicy (deprecated but may still exist in older clusters)
    - PodSecurityAdmission

8. Events & Metadata
    - Event
    - Lease

9. Extensibility
    - CustomResourceDefinition (CRD)
